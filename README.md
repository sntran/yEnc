The Erlang yEnc decoder and encoder.

## Installation

The package can be installed by adding `yEnc` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:yEnc, "~> 0.2.0"}
  ]
end
```

Documentation can be found at [https://hexdocs.pm/yEnc](https://hexdocs.pm/yEnc).
